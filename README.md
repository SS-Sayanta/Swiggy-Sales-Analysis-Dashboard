🍽️ Swiggy Sales Analysis Dashboard
📌 Project Overview

This project is an interactive Power BI dashboard built using a publicly available Swiggy dataset. It analyzes restaurant sales, customer behavior, and city-wise performance to generate meaningful business insights through interactive visualizations.

Note: This is an independent portfolio project created for learning and demonstration purposes. It is not affiliated with or endorsed by Swiggy.

🎯 Objectives
Analyze restaurant sales performance
Identify top-performing restaurants
Explore customer ordering trends
Compare city-wise sales
Build an interactive business dashboard for decision-making
